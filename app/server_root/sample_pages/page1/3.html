    このコンテンツは、Markdown形式でページを生成するサンプルです。

    Pickles Framework で Markdown を使用するには、ファイル名の最後に、拡張子 .md を追加します(例 index.html であれば、index.html.md)。

    どのように記述してこのHTMLがつくられているか、コンテンツのソースコードをみて確認してください。
