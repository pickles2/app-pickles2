Theme

テーマは、$conf->funcs->processor->html に、プロセッサー(加工処理)の1つとして設定されます。
そのため、テーマによって、テンプレートの実装方式が異なる場合があります。

標準的なテーマのテンプレートは、次のような規則で定義されます。

  • 標準的なテーマのテンプレートは、テンプレート格納フォルダの中に、{$layout}.html というファイル名で定義されます。
  • {$layout} の部分は、サイトマップCSVの layout 欄でページごとに切り替えることができます。
  • {$layout} には、次の種類が予め定義されています。
    • default (規定値)
    • top (トップページ用)
    • popup (ポップアップページ用)
    • plain (body要素が何もない)
    • naked (すべての要素をコンテンツが担当する)
  • 同じようにして、{$layout} はこれら以外にも自由に追加定義することができます。
